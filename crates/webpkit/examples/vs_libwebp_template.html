Method & caveats

In-process timing

libwebp linked via libwebp-sys and called in the same process — no CLI startup or file-IO skew. Median of warmed-up iterations.

Fairness

Encode is effort-matched (tier ↔ method) at q75 on identical pixels; decode is timed on each codec's own output (VP8 decode is bit-exact, so cross-decoding is redundant).

The SIMD gap

webpkit is safe scalar Rust by design. The decode gap is structural — closing it means autovectorization-friendly kernels, not a missed micro-op.

Reproduce

Regenerate anytime with just report-vs-libwebp. Wall-clock is machine/thermal-dependent, so this report lives in target/, never committed.
